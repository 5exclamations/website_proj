# BAK Azerbaijan Website

Tri-language landing page for Bioacoustic Correction service in Azerbaijan.

## Languages
- Azerbaijani (default)
- English
- Russian

## Run
Open `index.html` directly in the browser.

## Notes
Content is adapted from the referenced source page and localized for Azerbaijan.
This is an informational landing page and should be reviewed by a medical specialist before publication.

## Web3Forms setup
Client reviews, contact form, and booking modal submissions are wired to Web3Forms.

1. Create an access key at Web3Forms.
2. Open `script.js` and set:
	- `WEB3FORMS_ACCESS_KEY = "YOUR_REAL_KEY"`
3. Save and redeploy the static site.

Until the key is set, the forms stay visible but won’t send data.
